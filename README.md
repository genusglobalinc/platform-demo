# MVP Project
## Installation
1. Clone the repo: `git clone <repo-url>`
2. Backend:
   - `cd backend`
   - `pip install -r requirements.txt`
   - `uvicorn app:app --reload`
3. Frontend:
   - `cd frontend`
   - `npm install`
   - `npm start`
